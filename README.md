# SiWUB
Aplikasi ini merupakan Pengembangan Layanan pada bidang KUKM DISDAGKOPERIN kota Cimahi. Aplikasi Berbasis Web ini bertujuan untuk memberikan informasi seputar Disdagkoperin kota cimahi serta memberikan panduan bagi individu yang tertarik untuk memulai usaha baru. Platform ini menyediakan akses mudah dan cepat melalui aplikasi web dengan berbagai layanan yang diperlukan oleh para wirausaha, seperti pelatihan, konsultasi, dan sumber daya lainnya, Pengguna akan dapat mengakses berbagai konten yang relevan dengan kegiatan dan layanan yang disediakan oleh Disdagkoperin Cimahi, seperti informasi, berita, materi dan program-program yang tersedia yang ada didalamnya. Aplikasi ini memiliki beberapa fitur utama yitu log in dan register, WUB (learning system), berita, kontak, profile, absensi dan admin
